`DataTable` is a convenience wrapper on top of `DataView` that allows consumers to use the existing `DataTable` API, so you can have all the benefits
of `DataView` without needing to change any code.

To use this, just change your existing `DataTable` import to use the `@odyssey-react-mui/labs` version.

For example, this:

```
<DataTable
  getData={getData}
  columns={props.columns}
  hasChangeableDensity={props.hasChangeableDensity}
  hasColumnResizing={props.hasColumnResizing}
  hasColumnVisibility={props.hasColumnVisibility}
  hasSorting={props.hasSorting}
  initialDensity={props.initialDensity}
  renderDetailPanel={props.renderDetailPanel}
  rowActionButtons={props.rowActionButtons}
  rowActionMenuItems={props.rowActionMenuItems}
  {...props}
/>
```

is the same as:

```
<DataView
  availableLayouts={["table"]}
  getData={getData}
  tableLayoutOptions={{
    columns: props.columns,
    hasChangeableDensity: props.hasChangeableDensity,
    hasColumnResizing: props.hasColumnResizing,
    hasColumnVisibility: props.hasColumnVisibility,
    hasSorting: props.hasSorting,
    initialDensity: props.initialDensity,
    renderDetailPanel: props.renderDetailPanel,
    rowActionButtons: props.rowActionButtons,
    rowActionMenuItems: props.rowActionMenuItems,
  }}
  {...props}
/>
```

For more information, consult the DataView documentation.
